About the project

This research project develops and evaluates machine learning methods to forecast groundwater levels across France using in-situ observations, meteorological forcing, remote sensing indices, and geological constraints. The research emphasizes reproducibility, uncertainty quantification, and transferability between catchments.

Working packages

Three coordinated packages

WP1 — Data Collection & Processing

Assemble observational and exogenous datasets, perform quality control, and create standardized feature sets.

  • Objectives: collect national piezometric series, meteorological forcings (Météo-France), river flows, satellite indices (GRACE, Sentinel), and lithology maps.
  • Deliverables: cleaned time series, metadata catalogue, reproducible ETL pipelines.
  • Lead: C. Martin

WP2 — Modeling & Machine Learning

Develop candidate models, from classical time-series (ARIMAX) to modern methods (LSTMs, Temporal Convolutional Networks, Gaussian Processes, Physics-Informed ML).

  • Objectives: benchmark models, uncertainty estimates, feature importance, and generalization tests across regions.
  • Deliverables: model library, evaluation suite, reproducible notebooks.
  • Lead: L. Moreau

WP3 — Deployment & Monitoring

Productize top models, deploy APIs and dashboards, and design an operational monitoring pipeline with automated retraining triggers.

  • Objectives: dockerized services, CI/CD, model monitoring, stakeholder-ready visualizations.
  • Deliverables: REST API, web dashboard (interactive forecasts), deployment manual.
  • Lead: S. Lefèvre
